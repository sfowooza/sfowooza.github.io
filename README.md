# sfowooza.github.io

GitHub Pages site for **Ventures - Personal Finance Tracker**

## About

Ventures is a locally-installed personal finance application that helps you track:
- Income transactions and sources
- Expense records and categories
- Investment and savings account details
- Creditor and debtor information
- Budget targets and planning notes

## Key Features

- **100% Offline**: All data stored locally on your device
- **Privacy First**: No accounts, no data collection, no servers
- **Secure**: PIN protected with AES-256 encryption
